## 42.1 동기 처리와 비동기 처리
함수를 호출하면 함수 코드가 평가되어 함수 실행 컨텍스트가 생성된다. 이때 생성된 함수 실행 컨텍스트는 실행 컨텍스트 스택(콜 스택이라고도 부른다)에 푸시되고 함수 코드가 실행된다.
- 함수 코드의 실행이 종료하면 함수 실행 컨텍스트는 실행 컨텍스트에서 팝되어 제거된다.

```js
// [예제 42-01]
const foo = () => {};
const bar = () => {};

foo();
bar();
```
