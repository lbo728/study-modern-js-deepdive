## 27.3 length 프로퍼티와 희소 배열
- length 프로퍼티의 값은 배열에 요소 추가, 삭제 시 자동 갱신
```js
const arr = [1, 2, 3];
console.log(arr.length); // 3

arr.push(4);
console.log(arr.length); // 4

arr.pop();
console.log(arr.length); // 3
```

```js
// 희소 배열
const sparse = [, 2, , 4];
console.log(sparse.length); // 4
console.log(sparse); // [empty, 2, empty, 4]
// → 인덱스가 0, 2인 요소가 존재하지 않음
// → 희소
```
- 위의 배열에서 인덱스가 0, 2인 요소가 존재하지 않음
- 희소 배열의 length는 희소 배열의 실제 요소 개수보다 *언제나 크다*.
- 배열에는 같은 타입의 요소를 연속적으로 위치시키는 것이 최선!
  → 희소 배열을 생성하지 않도록 주의!

## 27.4 배열 생성
  ### 27.4.1 배열 리터럴
  - 가장 일반적이고 간편한 배열 생성 방식
  ```js
  const arr = [1, 2, 3];
  ```

  ### 27.4.2 Array 생성자 함수
  - 전달된 인수가 1개 & 숫자 → length 프로퍼티 값이 인수인 배열 생성
  - 이때 생성된 배열 = 희소 배열 → length 프로퍼티 값은 10, but 요소가 존재하지 X. 
  ```js
  const arr = new Array(10);

  console.log(arr); // [empty * 10]
  console.log(arr.length); // 10

  console.log(Object.getOwnPropertyDescriptors(arr));
  // { length: {value: 10, writable: true, enumerable: false, configurable: false} }
  ``` 

  ### 27.4.3 Array.of
  - 전달된 인수가 1개 & 숫자 → 인수를 요소로 갖는 배열 생성
  ```js
  Array.of(1); // [1]
  Array.of(1, 2, 3); // [1, 2, 3] 
  Array.of('string'); // ['string']
  ```

  ### 27.4.4 Array.from
  - *유사 배열 객체* or *이터러블 객체*를 인수로 전달받아 배열로 변환
  ```js
  // 유사 배열 객체
  Array.from({ length: 2, 0: 'a', 1: 'b' }); // ['a', 'b']

  // 이터러블 객체 (문자열)
  Array.from('Hello'); // ['H', 'e', 'l', 'l', 'o']

  // length만 있으면 요소는 undefined
  Array.from({ length: 3 }); // [undefined, undefined, undefined]

  // 두 번째 인수로 전달한 콜백 함수의 반환값으로 배열 구성
  Array.from({ length: 3 }, (_, i) -> i); // [0, 1, 2]
  ```

## 27.5 배열 요소의 참조
```js
const arr = [1, 2];

// 인덱스가 0인 요소를 참조
console.log(arr[0]); // 1
```

## 27.6 배열 요소의 추가와 갱신
```js
const arr = [1, 2];

// 배열 요소 추가
arr[2] = 100;

console.log(arr); // [1, 2, 100]
console.log(arr.length); // 3

// 배열 요소 갱신
arr[1] = 10;

console.log(arr); // [1, 10, 100]
```

## 27.7 배열 요소의 삭제
```js
const arr = [1, 2, 3];

delete arr[1];
console.log(arr); // [1, empty, 3]
console.log(arr.length); // 3

arr.splice(1, 1);
console.log(arr); // [1, 3]
console.log(arr.length); // 2
```
